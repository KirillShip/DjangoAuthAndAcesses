Проект на Django с кастомной системой авторизации и аутентификации. Реализована система доступа к различным ресурсам (Mock View), 
возможность изменения профиля, login, logout, регистрация и админская панель для выдачи доступов различным ролям.
В БД (PostgreSQL) созданы сущности User, Role, Permission, Element

Таблица доступа различных ролей на сайте к элементам:

* Отсутствие прав - 0
* Чтение - 1
* Создание - 2
* Обновление - 3
* Удаление - 4
